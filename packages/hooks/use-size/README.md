# @yamada-ui/use-size

> This is an internal utility, not intended for public usage.

## Installation

```sh
$ pnpm add @yamada-ui/use-size
```

or

```sh
$ yarn add @yamada-ui/use-size
```

or

```sh
$ npm install @yamada-ui/use-size
```
